## 1.0.0

- Initial release of IPSTACK VideoCall SDK
- Video/Audio calling with WebRTC via LiveKit
- Push notifications via Firebase Cloud Messaging (FCM)
- CallKit support for iOS (native VoIP calling experience)
- Custom VoIP notification for Android
- Background push handling for receiving calls when app is closed
- User credential storage with flutter_secure_storage for auto-login
- License-based access control
- SRTP/TLS 1.3/DTLS-SRTP encryption for secure communications
